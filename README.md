## ✨ Features

- 🔐 **User Authentication** - Secure login/signup with JWT
- 💻 **Online Code Editor** - Multi-language support with syntax highlighting
- ⚡ **Real-time Code Execution** - Execute code in isolated containers
- 🏆 **Competitions** - Create and join coding contests
- 📊 **Leaderboards** - Global and contest-specific rankings
- 📝 **Problem Set** - Curated coding challenges by difficulty
- 👥 **User Profiles** - Track progress and achievements
- 🌙 **Dark/Light Mode** - Customizable themes

## 🛠️ Tech Stack

### Frontend
- React.js / Next.js
- Tailwind CSS
- Monaco Editor
- Socket.io Client

### Backend
- Node.js / Express.js
- LocalStorage / SQL (Mocked)
- Redis (Caching)
- Docker (Code Execution)

### DevOps
- Docker & Docker Compose
- Nginx
- GitHub Actions (CI/CD)
